カード表示 — 3方向性の比較

カードサイズの切替UI(サイクルボタン)は決定済みという前提で、「カードの中身の見せ方」を3方向で比較する。 右上のサイズ切替ボタンは3案すべてに連動し、同じサイズでの表示差だけを比較できる。

kanban-terminal

サイズ切替(小/中/大)は3案すべてに連動する。テーマ切替は上部右端のモック確認用コントロール。

現状踏襲(密度のみ)
